# Hero Media Assets

이 폴더에 파일을 떨어뜨리면 앱이 자동으로 사용해요.

## hero.mp4 (IDLE 화면 헤로 비디오)

**규격:**
- 16:9 비율 (예: 1280×720, 1920×1080)
- 5~10초 loop
- H.264 / mp4, 무음
- ~2~5MB (너무 크면 첫 로드가 느려요)

**나노바나나 / Veo / Runway에 줄 prompt 예시:**

### Option A — 도심 야간 주행 (추천)
```
Dashcam POV driving through a nighttime city street, slow forward motion,
neon traffic lights reflecting on wet asphalt, cinematic, 6 second loop,
no sound, 16:9 aspect ratio, dark and moody atmosphere.
```

### Option B — 한국 도로 톤
```
First-person view from a car driving on a Korean city road at dusk,
gentle forward motion, soft headlights, traffic in the distance,
cinematic color grading, 8 second loop, no sound, 16:9.
```

### Option C — 고속도로 항공샷
```
Aerial top-down view of a highway interchange at sunset,
cars moving smoothly in multiple lanes, slow loop, cinematic,
8 seconds, no sound, 16:9.
```

생성된 mp4를 이 폴더에 `hero.mp4` 이름으로 저장하면 앱이 자동으로 사용합니다.
파일이 없으면 SVG 일러스트가 fallback으로 표시돼요.
